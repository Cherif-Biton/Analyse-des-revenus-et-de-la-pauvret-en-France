# Analyse des revenus et de la pauvreté en France

## Description
Projet réalisé dans le cadre du cours de **Logiciels Spécialisés en Statistique (R)**  
— Licence Statistique-Informatique (LSI), Semestre 4  
Université Nazi Boni, Bobo-Dioulasso.

Ce projet analyse les revenus et le taux de pauvreté en France à partir des données  
**Filosofi 2016** (niveau communal), en se concentrant sur un département attribué.

## Objectifs
- Explorer et manipuler les données de revenus et pauvreté
- Comparer les distributions nationales et départementales
- Produire des visualisations statistiques commentées
- Garantir la reproductibilité complète de l'analyse
